2025 goal: become fluent in Java


What is the purpose/goal of this blog?

Blogging as an accountability mechanism -- should post progress on a daily-basis

What is the expectation for these posts?

That minimal expectation is to prove that at least 30 mins of work was done.

Methodology & Andragogy:

Implement something. add the date as a comment. commit, push.
then create an entirely new, empty branch.
Then, re-implement the system/app - it could be
very similar to the first impl., or possibly very different.
even if it's mostly the same, it will likely take less time
to implement the time around too.

The best way to learn a software system, is by implementing
it from scratch.

For example, if we ant to learn Apache Spark, a good
project would be writing an implementation, from scratch
based on the available docs.
