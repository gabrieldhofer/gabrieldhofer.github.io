Learning Angular Stuff

Check Node version
node -v

Check npm version
npm -v

Install Angular
npm install -g @angular/cli

Add ng to the PATH
$ export ANGULAR_CLI=\
$ export PATH=$PATH:$ANGULAR_CLI

Check Angular version
ng --version

Create project
ng new PROJECT_NAME

Build and run app
ng serve -o

Create new GitHub Repository
gh repo create \
