Creating and Deleting branches with Git

Original Post Date: 2025-02-19
Last Modified: 2025-02-19

Creating new branch 
				
	git checkout -b {new_branch_name} {based_on_branch}
Push branch 
				
	git push --set-upstream origin {branch_name}
User defined shell function, sets branch_name to current branch 
				
	gpsuo ()
	{
		BRANCH_NAME="$(git branch | grep \* | awk '{print $2}')";
		echo "current BRANCH_NAME: $BRANCH_NAME";
		git push --set-upstream origin $BRANCH_NAME
	}
Deleting branch locally 
				
	git branch -d {local_branch} [-D]
Deleting branch remotely 
				
	git push origin -d {remote_branch}
