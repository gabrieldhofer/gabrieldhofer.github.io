Creating live bootable USB drive on Windows

sources:
source 1 -- MS blog
source 1 -- Geeks for Geeks

1. Run the diskpart program 
             $ diskpart.exe


2. List the disks 
             $ list disk


3. Select disk of the USB drive
             $ select disk <USB-disk-number>


4. Create primary partition
             $ create partition primary


5. Select the partition that was just created: 
             $ select partition <partition-number>


6. Make the partition active..
             $ active


7. Leverage Rufus to create bootable disk
